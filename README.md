# AI Career Mentor (LLM Powered)

An intelligent career guidance web application that analyzes a user's resume and provides:

* Best suited job role prediction
* ATS Resume Score
* Skill gap analysis
* Personalized learning roadmap
* AI career chatbot guidance

---

## Features

* Resume parsing (PDF)
* Machine Learning role prediction
* ATS scoring system
* Semantic search using FAISS
* LLM powered career assistant (Groq API)
* Interactive Streamlit UI

---

## Tech Stack

Python, Streamlit, Machine Learning, Sentence Transformers, FAISS, LLM (Groq API)

---

## How to Run

```bash
git clone https://github.com/Misbashaikh0602/AI-Career-Mentor.git
cd AI-Career-Mentor
pip install -r requirements.txt
streamlit run app.py
```

---

## Project Workflow

Upload Resume → AI Analysis → Skill Gap → ATS Score → Roadmap → Chatbot Guidance
